# NPFO

Antifascist brawler prototype. Pure browser build served from `/docs`.

## Quickstart
1. Serve `docs/` statically (e.g. `cd docs && python -m http.server`).
2. Open `http://localhost:8000/` in a browser.

## Controls
- Move: **←/→**
- Jump: **↑**
- Jab: **Z**
- Heavy: **X**
- Dash: **C**
- Rebel surge: **V** (when meter full)
- Pause: **P**
- Debug HUD: **D**
