# NPFO

## Quickstart

Serve the project and open the game:

```bash
python3 -m http.server
```

Then visit [http://localhost:8000/docs/](http://localhost:8000/docs/).

## Controls

- **Left / Right**: move
- **Up**: jump
- **Z**: jab
- **X**: heavy
- **C**: dash
- **V**: Rebel surge
- **P**: pause
- **D**: debug HUD
